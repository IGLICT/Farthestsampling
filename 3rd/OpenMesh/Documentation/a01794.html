Detailed Description

template<class Refs_>
class OpenMesh::Concepts::MeshItems::VertexT< Refs_ >

Interface for the internal vertex type.

This minimal interface must be provided by every vertex. It's up to the mesh kernel (or the items used by the mesh kernel) to implement it.

All methods marked as internal should only be used by the mesh kernel.
